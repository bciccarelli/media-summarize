# Social Media Summarizer

I thought this would make a cool linkedin post but then i found out someone did basically the same thing already, so here you go.

## The idea: Filter out signal from noise

Summarizes your X and LinkedIn feed with local browser automation and sends you an email.  I've set up the report to run weekly. Requires gemini API key and you might want to tell claude or whomever to change the summary prompts to reflect your interests if you don't want AI/startup news.
